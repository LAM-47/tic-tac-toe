# tic-tac-toe
🎮 A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript with win detection, reset functionality, and clean UI.

This project demonstrates DOM manipulation, event handling, game logic implementation, and UI state management in JavaScript.

## 🚀 Features

- 🔄 Turn-based gameplay (X and O)
- 🏆 Automatic winner detection
- 🎉 Winner announcement message
- 🔁 Reset & New Game functionality
- 🚫 Prevents overwriting moves
- 💻 Clean and responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🧠 What I Learned

- Working with querySelectorAll
- Handling click events
- Game state management
- Implementing win pattern logic
- DOM manipulation
- Debugging real-world issues
